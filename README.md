# Valentine--
# 💌 Valentine's Day Proposal

A cute, interactive "Will You Be My Valentine?" webpage with:

- Romantic envelope opening animation  
- Typewriter intro effect  
- Floating stars, pulsing heart background  
- Playful "No" button that runs away + sad puppy GIF  
- Confetti, falling hearts, blooming flowers celebration when they say **Yes**  
- Mobile responsive design  


## ✨ Features

- Dramatic envelope opening sequence with sparkles  
- Animated gradient title & glowing text  
- Hover → "No" button escapes around the screen with funny sad messages  
- Switches to sad puppy GIF when "No" is hovered  
- Massive confetti + hearts + flowers explosion on "Yes"  
- Pure HTML + CSS + vanilla JavaScript (no framework)  
- Tailwind CSS CDN for utility classes + lots of custom animations


## 🚀 Demo

You can try it right now:
→ https://titanium-bullet.github.io/valentine-project/  
Or just open `index.html` in any modern browser — no build step needed!


## 🛠️ Technologies

- HTML5  
- CSS3 (animations, gradients, transforms, keyframes)  
- Tailwind CSS (via CDN)  
- Vanilla JavaScript  
- Google Fonts (Dancing Script + Poppins)
